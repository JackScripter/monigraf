# monigraf
Monitor Linux server and more !

## Installation
### Dependencies
* python3.7
* python3-pip
* inotify-tools

### Installation
```bash
git clone https://github.com/JackScripter/monigraf.git
cd monigraf
sudo bash install.sh
```
